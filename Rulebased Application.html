Dynamic Rule-Based Application Documentation

Introduction

Welcome to the Dynamic Rule-Based Application! This documentation will guide you on using the application to create, modify, and manage rules that determine eligibility based on user attributes. The app allows administrators to build rule sets from scratch and create flexible, dynamic workflows.

Getting Started

To start using the application, log in as an administrator and access the Dashboard. Here you will find options to create rules, manage existing rules, and view eligibility results.

  • Log in with your administrator credentials.
  • Navigate to the Dashboard from the main menu.
  • Click on Create New Rule to begin setting up a rule-based flow.

Creating Rules

The application allows you to create rules by defining conditions based on user attributes such as age, income, and department. Follow these steps to set up a new rule:

  • Click on Create New Rule in the Dashboard.
  • Select the attribute to base the rule on (e.g., Age, Income).
  • Define conditions (e.g., Age > 18, Income < 50000).
  • Click Save Rule to store the rule for future use.

Each rule can have multiple conditions, allowing for complex decision trees.

Combining Rules

To create dynamic workflows, you can combine multiple rules. This feature allows you to develop complex rule chains that evaluate various conditions simultaneously.

  • In the Rule Combination section, select rules you want to combine.
  • Define the relationship (AND/OR) between rules.
  • Preview the combined rule to ensure it matches your intended logic.
  • Click Save Combination to apply.

Modifying Existing Rules

You can edit existing rules from the Rule Management section. This allows you to adjust conditions, rename rules, or remove rules as your requirements change.

  • Select the rule you wish to modify.
  • Adjust the attributes or conditions as needed.
  • Click Save Changes to update the rule.

Note: Changes to rules may affect existing workflows and eligibility checks.

Running Eligibility Checks

With your rules in place, you can start performing eligibility checks for users. This feature evaluates a user's attributes against the rule set to determine eligibility.

  • Go to the Eligibility Check section.
  • Select a user or input their attributes manually.
  • Click Run Check to get a detailed eligibility report.

The report provides a summary of which rules were met or failed, offering a clear picture of the eligibility result.

Frequently Asked Questions (FAQs)

Can I delete a rule?

Yes, you can delete a rule by selecting it in the Rule Management section and clicking Delete. Be aware that deleting a rule may affect combined rules and eligibility checks.

How do I add more attributes for rule creation?

Additional attributes can be added in the Settings section under Manage Attributes. Here you can add or remove attributes to meet specific requirements.

What if I need help with a complex rule combination?

Contact our support team at support@example.com, or refer to the advanced guide in the Help section for more information on complex rule setups.
